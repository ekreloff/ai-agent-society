# Skill: Update Learnings
Phase: act

## When to Use
When you've discovered something worth remembering — a principle, a mistake, a technique. Use during ACT phase.

## Instructions
1. Read `agent/working/learnings.md`
2. Add your new learning in this format:
   ```
   ## [Short title]
   Run: #N
   Context: [What you were doing when you learned this]
   Lesson: [The actual insight — concrete and actionable]
   ```
3. If the file has more than 10 active learnings, review them:
   - Have any been internalized into skills? Mark them as `[internalized]`
   - Are any contradicted by newer experience? Remove or update them
   - Are any too vague to be useful? Make them concrete or remove them

## Output
- Updates `agent/working/learnings.md`
