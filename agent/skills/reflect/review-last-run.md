# Skill: Review Last Run
Phase: reflect

## When to Use
Every run, as the first thing you do. Skip only if this is run #1 (no history).

## Instructions
1. Read `agent/working/recent.md`
2. Read the most recent file in `agent/archive/runs/` (use Glob to find it)
3. Ask yourself:
   - What did I try last time?
   - Did it work? What was the score?
   - What did I say I'd do next?
   - Is there anything I was confused about?
4. Hold these answers in mind for the PLAN phase

## Output
No file changes. This skill produces orientation — understanding of where you are.
